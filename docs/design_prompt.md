# 디자인 프롬프트 — 프롬프트 슬롯머신

Claude Design에 그대로 붙여넣으세요.
디자인 시스템(색·타이포·컴포넌트)을 뽑는 것이 목적이며, 기능 구현은 별도로 진행합니다.

---

## 붙여넣을 프롬프트

```
AI 이미지 생성용 프롬프트를 무작위로 조합해주는 웹 도구의 디자인 시스템을 만들어주세요.
컨셉은 현대 카지노 슬롯머신 캐비닛입니다.

## 무엇을 하는 도구인가

11개 항목(장르, 배경, 인물, 조명, 앵글 등)을 무작위로 뽑아 조합한 뒤,
그 결과를 영문 프롬프트 문자열로 출력합니다.
사용자는 나온 결과를 복사해 다른 이미지 생성 도구에 붙여넣습니다.
마음에 드는 항목은 잠그고 나머지만 다시 돌릴 수 있습니다.

## 시각적 방향

2000년대 이후 현대 카지노 슬롯머신 캐비닛을 참조합니다.
- 대형 스크린, 다중 페이라인 그리드
- 물리 레버는 없습니다. 크고 명확한 버튼만 씁니다.
- 젤 버튼 특유의 볼록한 입체감, 눌리는 느낌
- 당첨 연출: 릴 정지 시 빛 번짐, 잠긴 릴의 강조 표시

명확히 피해야 할 것:
- 보라-파랑 그라디언트
- 유리질 반투명(glassmorphism)
- 과하게 둥근 코너
- 중앙 정렬된 히어로 섹션
- 일반적인 SaaS 대시보드 톤

## 레이아웃

5열 × 3행 그리드가 화면의 중심입니다. 릴은 전부 가로로 회전합니다.

행1: 장르 / 배경 / 소품 / 조명 / 앵글
행2: 주체유형 / 성별 / 연령 / 역할 / 종족
행3: 인물수 / (빈칸 4개)

주체유형 릴의 값에 따라 행2의 오른쪽 4칸이 켜지거나 꺼집니다.
꺼진 릴은 어둡게 비활성 상태로 표시되어야 합니다. 이 상태 디자인이 중요합니다.

그리드 아래에 결과 프롬프트 출력 영역, 그 아래 컨트롤 영역이 옵니다.

## 필요한 컴포넌트

1. 릴 셀
   - 상태: 대기 / 회전중 / 정지 / 잠김 / 비활성
   - 각 셀에 잠금 토글과 개별 재추첨 버튼
   - 셀 안에 영문 값과 한글 해설이 함께 들어감 (영문이 주, 한글이 보조)
   - 값의 길이가 제각각임 ("elf"부터 "one figure in foreground, many behind"까지)

2. 메인 버튼 (전체 재추첨)
   - 젤 버튼, 크고 명확함
   - 화면에서 가장 강한 요소

3. 결과 출력 영역
   - 영문 프롬프트 전문, 복사 버튼
   - 부정 프롬프트 별도 필드
   - 한글 해설 영역
   - 출력 개수가 여러 개일 때 목록으로 쌓임

4. 컨트롤
   - 슬라이더 (가중치 조절, 물리/연출 비율)
   - 드롭다운 (화면비 10종, 출력 개수)
   - 토글 스위치 (종족·장르 충돌 필터)
   - 텍스트 입력 (접두사·접미사 편집)

5. "건짐" 표시 버튼
   - 마음에 든 조합을 기록하는 용도
   - 슬롯머신의 당첨 표시 같은 느낌

6. 로그 화면
   - 기록된 조합 목록, 항목별 적중률 통계

## 색

아래 3가지 방향을 각각 제시해주세요. 그중 고르겠습니다.

방향 1 — 골드 & 레드
전통적인 카지노 배색. 짙은 적색 캐비닛, 금색 트림, 검정 스크린.

방향 2 — 네온 (마젠타/시안)
전자식 캐비닛 계열. 어두운 바탕에 고채도 네온 액센트.

방향 3 — 크롬 & 앰버
금속성 회색 바탕에 호박색 라이트. 앞의 둘보다 절제된 방향.

각 방향에 대해 배경/표면/텍스트/액센트/상태색(잠김·비활성·당첨) 팔레트를 정의해주세요.

## 타이포그래피

- 릴에 들어가는 영문 값: 가독성이 최우선. 폭이 좁고 길이 편차를 감당해야 함
- 숫자 표시(카운터, 확률): 7-세그먼트 또는 그에 준하는 기계적 서체
- 한글 해설: 영문보다 작고 약하게, 그러나 읽을 수 있게
- 라벨: 각인된 느낌. 대문자, 자간 넓게

## 톤

화려하되 조잡하지 않게. 실제 도구로 매일 쓸 물건입니다.
연출은 릴이 도는 순간과 당첨 표시에만 집중하고,
나머지 시간엔 조용해야 합니다.
```

## 프롬프트 작성 시 고려한 것

1. **피해야 할 것을 명시** — "AI스러운 디자인"의 정체가 특정 기본값 조합이므로 그걸 직접 나열했습니다.
2. **값 길이 편차를 명시** — 릴 셀 디자인의 최대 난점입니다. 안 짚으면 짧은 단어 기준으로 디자인됩니다.
3. **비활성 상태를 강조** — 주체유형 분기 때문에 릴이 꺼지는데, 이게 이 도구 고유의 문제라 짚어뒀습니다.
4. **색은 후보 3개** — 하나로 정하지 않고 고르실 수 있게 했습니다.
5. **"조용해야 한다"** — 카지노 컨셉이 과열되면 매일 못 쓰는 물건이 됩니다.
